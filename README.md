# INSTALLATION

Clone this repository: 

```
https://github.com/stephanebtvr/marvel-frontend.git
cd marvel-frontend
```

Install packages:

 ```
 yarn install
 ```

When installation is complete, launch the project:

 ```
 yarn start
 ```
